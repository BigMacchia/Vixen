What is vixen?
Vixen is an asset management system for maya.
It's in early beta mode and mostly undocumented.

NOTE: Read license.htm before installing and using Vixen.


Install:
copy vixen folder and userSetup.py file to your Maya scripts folder.


How to use it?
Vixen is availabe via channel sidebar. Just check in your files after changing them.
